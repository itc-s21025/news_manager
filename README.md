#  サーバーサイドkotlinの課題制作  

## NewsManagerの仕様
* ニュースの一覧所得可能 (認証なしでも所得可)
* ニュースの登録 
* カテゴリーの追加 
* カテゴリーの更新 
* カテゴリーの削除 
* 管理者、ユーザーの認証 

-----------------------------------------------------------
*データベースのテーブルを作成する SQL と初期データは別途掲載 
*ニュースは、概要のリストと1記事の詳細で異なるエントリーポイントとする 
*ニュースは認証(ログイン)無しで取得が可能 
*ニュースの投稿は認証ユーザだけができる 
*ユーザの追加は管理者権限を持つユーザだけができる 
*カテゴリの追加・変更・削除は認証ユーザだけができる
